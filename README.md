# 🪔 Diwali Sales Analysis (Python Project)

> End-to-end exploratory data analysis of Diwali sales data to uncover customer behavior, sales trends, and business insights.

---

## 📌 Overview

This project analyzes Diwali sales data using Python to understand customer purchasing behavior and generate actionable business insights.

---

## 🎯 What This Project Explores

| Area                 | Description                                |
| -------------------- | ------------------------------------------ |
| 📊 Sales Analysis    | Overall revenue and purchasing trends      |
| 👥 Customer Analysis | Gender, age group, and occupation insights |
| 🛍 Product Analysis  | Top-selling products and categories        |
| 🌍 Regional Analysis | State-wise sales performance               |
| 📈 Trend Analysis    | Buying patterns across different segments  |

✔ A complete **EDA (Exploratory Data Analysis)** process was performed.

---

## 🛠 Tools & Technologies

| Tool       | Purpose                      |
| ---------- | ---------------------------- |
| Python     | Data analysis                |
| Pandas     | Data cleaning & manipulation |
| Matplotlib | Data visualization           |
| Seaborn    | Advanced visualizations      |

## 📂 Dataset Description

| Column           | Description            |
| ---------------- | ---------------------- |
| User_ID          | Unique customer ID     |
| Gender           | Customer gender        |
| Age Group        | Customer age category  |
| Occupation       | Customer profession    |
| Product_Category | Purchased product type |
| Amount           | Purchase amount        |
| State            | Customer location      |

## 📈 Key Insights

* Female customers contribute a higher share of total sales
* Age group 26–35 contributes the most to overall revenue
* Certain states generate maximum sales volume
* A few product categories dominate total revenue

---

## 📌 Business Recommendations

* Target female customers aged 26–35 with personalized campaigns
* Focus on high-performing product categories for inventory planning
* Increase marketing efforts in top-performing states
* Use seasonal trends to optimize sales strategy
